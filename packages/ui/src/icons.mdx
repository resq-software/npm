# Icons

`@resq-sw/ui` uses [Phosphor Icons](https://phosphoricons.com) (`@phosphor-icons/react`) for all iconography.

## Default weight

All icons inside library components use `weight="light"` (1.5px stroke) for a refined, minimal aesthetic. This is intentional — it differentiates ResQ from the default shadcn look.

## Usage

All 1512 Phosphor icons are re-exported from `@resq-sw/ui/icons` (CSR) and `@resq-sw/ui/icons/ssr` (RSC / Next.js App Router):

```tsx
// CSR — client components, vanilla React
import { CheckIcon, CaretDownIcon } from "@resq-sw/ui/icons";

// SSR — React Server Components, Next.js App Router
import { CheckIcon, CaretDownIcon } from "@resq-sw/ui/icons/ssr";

<CheckIcon className="size-4" weight="light" />
```

You can also import directly from Phosphor if you prefer — both are valid.

## Weight system

Phosphor provides six weights per icon — use them intentionally:

- **`thin`** (1px) — Decorative, background elements
- **`light`** (1.5px) — **Library default** — UI chrome, navigation
- **`regular`** (2px) — Fallback when more visual weight is needed
- **`bold`** (3px) — Emphasis, active states
- **`fill`** (solid) — Selected/active indicators, toggles
- **`duotone`** (two-tone) — Decorative illustrations, empty states

See the **Weights** story for a visual comparison.

## Sizing

Use Tailwind size utilities, not the `size` prop:

```tsx
{/* Correct */}
<CheckIcon className="size-4" weight="light" />

{/* Avoid */}
<CheckIcon size={16} weight="light" />
```

Common sizes: `size-3` (12px), `size-3.5` (14px), `size-4` (16px), `size-5` (20px), `size-6` (24px).

## Adding new icons

The barrel already covers the full Phosphor set (1512 icons, `@phosphor-icons/react` v2.1.10). When Phosphor releases new icons, add them in three places:

1. **`src/components/icons/index.ts`** — CSR barrel (imports from `@phosphor-icons/react`)
2. **`src/components/icons/ssr.ts`** — SSR mirror (same icons, imports from `@phosphor-icons/react/ssr`)
3. **`src/components/icons/icons.stories.tsx`** — import block + registry entry with appropriate tags
